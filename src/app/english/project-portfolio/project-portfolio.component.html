Portfolio Website

Description

This project is the Website you are currently viewing. It was created to showcase my experience and skills in web development. While I am not a designer and merely an engineer, I tried to make it look as good as I can.

Tech Used

This app uses Angular to build the front end. The styling is done with Sass without the use of a framework. This project's source code is hosted on Github. I also use Github Actions to automate the deployment when I merge to the master branch. This project is hosted on the Firebase platform. The deployment is automated via Github.
